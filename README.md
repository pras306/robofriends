## How To Use

The website is published at [Robofriends](https://pras306.github.io/robofriends/)

Clone the project 

```bash
  git clone https://github.com/pras306/robofriends.git
```

Go to the project directory

```bash
  cd robofriends
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
